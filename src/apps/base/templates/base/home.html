Система управления заказами в кафе

Простое и удобное решение для управления заказами в кафе. Платформа помогает эффективно обрабатывать заказы, отслеживать их статусы и анализировать выручку.

Что умеет платформа?

  • Добавление заказа:

    Через веб-интерфейс вы можете добавить заказ, указав номер стола и список блюд с ценами. Платформа автоматически присвоит заказу уникальный ID, рассчитает общую стоимость и установит статус "в ожидании".

  • Удаление заказа:

    Вы можете удалить заказ, выбрав его по ID через веб-интерфейс. Это удобно для отмены ошибочных или завершенных заказов.

  • Поиск заказа:

    Платформа позволяет искать заказы по номеру стола или статусу. Это помогает быстро находить нужные заказы и управлять ими.

  • Отображение всех заказов:

    На отдельной странице отображается таблица со всеми заказами. Вы можете видеть ID заказа, номер стола, список блюд, общую стоимость и текущий статус.

  • Изменение статуса заказа:

    Вы можете изменить статус заказа на "в ожидании", "готово" или "оплачено". Это помогает отслеживать процесс выполнения заказов.

  • Расчет выручки за смену:

    Платформа автоматически рассчитывает общую выручку за смену на основе оплаченных заказов. Это удобно для анализа и отчетности.

REST API

Для интеграции с другими системами доступен REST API по адресу:

/orders/api/v1

API поддерживает все основные операции: создание, чтение, обновление, поиск и т. д.
